## 2. 목표
- 회의 일정을 작성하여 회의실 사용 현황을 조직 사람들에게 공유한다.

## 3. 사용자 유형
- 사용자:
    - 회의 일정을 작성한다.
    - 사용자는 작성한 회의 일정을 볼 수 있다.

## 4. 사용자 스토리
- 사용자:
  - 조직에서 계정을 하나 만들고 공유한다.
  - 공유된 계정으로 조직 사람들이 회의 일정을 공유한다.

| ID  | 기능 요구사항     | 설명                           | 우선순위        |
